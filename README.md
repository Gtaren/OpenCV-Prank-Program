# OpenCV-Prank-Program

This program bring you fake hacked popup and shut down your PC.

## Prerequisites

- Python 3.x
- Required libraries
  - numpy, pyautogui, opencv-python
- usable OS
  - Windows, Linux

## Usage
1. Install required libraries.
2. Change images to what you like. (Initial images: my lovely dog)
3. Image names should be **(number).jpg**.
4. Try it. (Be sure that your PC will shut down)
```
python3 prank.py
```
